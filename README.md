# ☕ Java – Herança, Polimorfismo e Interfaces Practice (Módulo 11)

Este repositório contém **exercícios e exemplos práticos** relacionados aos pilares da **Programação Orientada a Objetos em Java**:  
✔ Herança  
✔ Polimorfismo  
✔ Interfaces  

Esses exercícios fazem parte do **Módulo 11 do curso na plataforma Alura** e foram organizados para reforçar a aplicação desses conceitos no código.

---

## 📚 Conteúdos práticos

Os principais tópicos revisados e praticados neste módulo incluem:

### 🔹 Herança
- Reutilização de código entre classes (classe pai → classe filha)
- Estendendo funcionalidades de classes base para especializações

### 🔹 Polimorfismo
- Chamadas de método que se comportam de formas diferentes conforme a instância
- Polimorfismo de sobrescrita (`@Override`) em subclasses

### 🔹 Interfaces
- Definição de contratos que as classes implementam
- Uso de interfaces para padronizar comportamentos

Esses pilares são essenciais para organizar classes e objetos em aplicações Java mais robustas. :contentReference[oaicite:1]{index=1}
